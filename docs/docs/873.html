Free PowerShell Commands for Active Directory

May 11, 2011 • kmhuglen

As I have been using a lot of PowerShell script lately for extracting and modifying Active Directory objects, and of course search the web for how-to and examples on how to do stuff. I have often come over examples where they use CMDLETS that’s not available from a the default Active Directory Module for Windows PowerShell.

After a couple of new google search I found

ActiveRoles Management Shell for Active Directory from Quest Software. It is free and you can download it from this site: http://www.quest.com/powershell/activeroles-server.aspx
